# UHF_AndroidReader

#### 介绍
102s固定式rfid服务项目


#### 软件模块介绍
AndroidReader  服务模块，通过aidl与sdk通信  完成rfid相关功能
readSdk   sdk模块，模块内已经完成了aidl的连接服务，只需要监听Constants.connectState即可，简单使用方法在testDemo中有相应代码
app       UI界面展示


#### 使用说明

1. 安装AndroidReader服务apk
2. 项目中载入sdk依赖
3. run 项目
